BWH - Terminology Glossary

A short guide to terminology used on this Site:

A list of common and not-so-common terms used on this Site and many others. It is by no means intended to be an exhaustive listing and its purpose is as a guide rather than rule.

Apple Mac
~computing
Apple Incorporated, a Hardware and Software Company est. in 1983. The underlying Operating System - OS (Darwin) now being based on FreeBSD Linux.
Bespoke
~adjective
Made to order, customised etc. Used to describe on this Site that a service we offer can be customised to suit your needs beyond the scope of its original parametres.
Blog
~noun
A shortening of Weblog. A WebSite feature in which an individual or group produces an ongoing narrative.
Broadband
~computing
Signals over a wide range of frequencies in high-capacity TeleCommunications allowing for faster dowload rates than most dial-up options.
Browser
~computing
Programme with a Graphical User Interface - GUI for displaying HTML (or similar Markup/Meta language) files, used to navigate the World Wide Web - WWW, also known as a User Agent - UA: a Web Browser | a Web User Agent.
Code|d
~computing
Programme instructions: hundreds of lines of code | coded in perl.
Cookies
~computing
A packet of data sent by an Internet server to a Browser, which is returned by the Browser each time it subsequently accesses the same server, it is used to identify the User.
Daemon
~noun | computing
A background process that handles requests for services such as print spooling and file transfers, remains dormant when not required.
Linux
~computing | trademark
An Open Source version of the UNIX Operating System. Free|Open|Net|BSD, Debian, RedHat, SlackWare, Knopix & Ubunto being just a few of the current flavours.
Login|out
~computing
An act of logging into or out of a Computer system, usually allowing backend access and/or further information & privileges.
Lynx
~computing
A fully-featured World Wide Web - WWW client for users running cursor-addressable, character-cell display devices (e.g., vt100 terminals, vt100 emulators running on Linux, PCs or Macs, or any other character-cell display).
Markup
~computing
A set of tags assigned to elements of a text to indicate their structural or logical relation to the rest of the text. SGML, HTML & XML being some examples.
Open Source
~computing
Computer Software whose source code is put into the public domain, subject to the restriction that any improvements or derived Software also include the source code and be put back into the public domain.
Operating System
~computing
Software that supports a Computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
perl
~computing
Created by Larry Wall, perl is a common and popular programming language often mistakenly refered to as the “Practical Extraction & Reporting Language” or the “Pathologically Eclectic Rubbish Lister”. Neither are correct (as it is not an acronym) and is sometimes known as Perl (uppercase P), it is also closely associated with a camel… don't ask…
Pop-Under
~computing
Relating to, or denoting an additional window, usually an advertisement, that is under a Web Browsers main or current window and appears when a user tries to exit: a plague of flashing pop-under ads. See also 'Pop-Up'.
Pop-Up
~computing
A menu or other utility able to be superimposed on a Web Browsers main or current window. See also 'Pop-Under'.
TeleConference
~noun
A conference with participants in different locations linked by TeleCommunications devices.
UNIX
~computing | trademark
A widely used multiuser Operating System - OS, IRIX, Solaris & SunOS being three popular variants. Origin: 1970s, from uni- [one] + a respelling of -ics , on the pattern of an earlier less compact system called Multics.
User Agent
~computing
Another term for a Browser - See Browser.
Username
~computing
An identification used by a person with access to a Computer Network. A password is usually required in conjunction to allow a Login.
VideoConference
~noun
An arrangement in which television sets, VideoPhones, or Computers are used to enable a group of people in several different locations to communicate with each other in sound and vision.
