BWH - Site Acronyms & Abbreviations

A guide to acronyms & abbreviations used on this Site:

The Internet and Computer technology in general are inundated with acronyms and abbreviations, following is a collection that you will find used on this Site and many others.

ADSL
Asynchronous (or Asymmetric) Digital Subscriber Line
A method of routing digital data on copper telephone wires, allowing high-speed Internet access and simultaneous use of the line for voice transmission.
AJAX
Asynchronous JavaScript And XML
It is the use of the nonstandard XMLHttpRequest object to communicate with server-side scripts. Its most appealing characteristic, however, is its “asynchronous” nature, which means it can do all of this without having to refresh the page.
AS
Apple Safari
Apple Computer Inc. The Apple Mac Safari Browser/UA.
CLI
Command Line Interface
A programme that a user enters data or commands for a Computer System which then displays the received output.
CMS
Content Management System
An Online or Local Area Network - LAN means of administering data by use of a Graphical User Interface - GUI.
CSS
Cascading Style Sheets
A document (or documents) that controls the behaviour of defined HTML elements.
CPU
Central Processing Unit
The part of a computer in which operations are controlled and executed.
CSV
Comma Seperated Values
A document with text that is used to transfer data from one programme to another with the fields seperated by commas.
DBA
DataBase Administrator
A person who is responsible for the planning, creation and administration of Databases.
DHTML
Dynamic HyperText Markup Language
A collection of Web Browser enhancements that enable dynamic and interactive features on Web pages.
FF
FireFox
Mozilla Firefox. The Mozilla Foundation's Browser/UA.
GPL
General Public License
A basic legal document that covers Open Source Software.
GPRS
General Pocket Radio Services
A technology for radio transmission of small packets of data, especially between cellular phones and the Internet.
GUI
Graphical User Interface
A visual way of interacting with a Computer using items such as windows, icons, and menus, used by most modern Operating Systems - OSs.
HTML
HyperText Markup Language
A standardised system for tagging text files to achieve font, colour, graphic, and HyperLink effects on World Wide Web - WWW pages.
HTTP
HyperText Transport Protocol
HyperText (or Transfer) Protocol, the data transfer protocol used on the World Wide Web - WWW.
IRC
Internet Relay Chat
An area of the Network or Web where users can communicate interactively with each other.
IM
Instant Messanger
See IRC.
ISP
Internet Service Provider
The people who connect you to the Internet.
IT
Information Technology
The study or use of systems (esp. Computers and TeleCommunications) for storing, retrieving, and sending information.
KK
KDE's Konqueror
KDE's Konqueror Browser/UA.
LAN
Local Area Network
A Computer Network that links devices within a building or group of adjacent buildings. See also - WAN.
MSIE
MicroSoft Internet Explorer
Microsoft's Browser/UA.
NDA
Non Disclosure Agreement
A legal document that is drawn up between two parties which outlines an agreement not to divulge any information pertaining to any project or venture that either party is working on contained in the document.
OSO
Opera Software Opera
Opera Software's Browser/UA.
OS X
Operating System X
Apple Mac's version 10 Operating System - OS.
PDA
Personal Digital Assistant
A palmtop computer used to store information such as addresses and telephone numbers, and for simple word processing and spreadsheeting.
PHP
PHP: Hypertext Preprocessor
A widely-used open source general-purpose scripting language that is especially suited for Web development and can be embedded into HTML.
PGP/GPG
Pretty Good Privacy/Gnu Privacy Guard
Software that allows military grade encryption of private data.
POP
Point Of Presence
Denotes equipment that acts as an access to the Internet. Also known as PoP.
PPP
Point to Point Protocol
Allows data conforming to the Internet Protocol - IP to be handled on a serial line.
RFC
Request For Comment
A document circulated on the Internet that forms the basis of a technical standard.
SFTP
Secure File Transfer Protocol
A secure standard for the exchange of programme and data files across a Network or the Internet.
SLA
Service Level Agreement
Is that part of a service contract in which a certain level of service is agreed. An SLA is therefore not a type of service contract, but a part of a service contract. A service contract can contain zero, one or more SLAs. A contract containing SLAs is usually referred to as a “performance contract”.
SMTP
Simple Mail Transfer Protocol
A data transmission format used to send and receive eMail.
SQL
Structured Query Language
An international standard for Database manipulation. MySQL, PostGres & Oracle being three of the most common variants.
SSH
Secure SHell
A programme for logging into and executing commands on a remote machine which provides secure encrypted communications between two untrusted hosts over an insecure Network.
SSL
Secure Sockets Layer
A computing protocol that ensures the security of data sent via the Internet by using military grade encryption.
SVr4
System V revision 4
System Five (V) Revision Four is a UNIX standard.
TCP/IP
Transport Control Protocol/Internet Protocol
Used to govern the connection of Computer systems to the Internet.
UA
User Agent
Another term for a Browser.
VoIP
Voice over Internet Protocol
A communications protocol that allows for telephonic communication via the Internet.
WAN
Wide Area Network
A Computer Network in which the Computers connected may be far apart, generally having a radius of half a mile or more. See also - LAN.
WWW
World Wide Web
A world wide network of interlinked Computers, Servers & Networks.
XHTML
eXtensible HyperText Markup Language
A HTML system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web - WWW pages, incorporating user-defined elements.
XML
eXtensible Markup Language
A Meta-Language that allows users to define their own customised markup languages, especially in order to display documents on the World Wide Web - WWW.
